AgenDAV is an open source multilanguage CalDAV web client which features a rich AJAX interface with shared calendars support
Some features:
  • Drag&Drop user interface
  • Multilanguage and timezone support
  • Multiple calendar support
  • Configurable calendar colours
  • Calendar sharing
  • Basic recurring events support
